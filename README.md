# Personal Schedule/Appointment Management System

## Project Description
A web-based appointment management system designed for dental clinics to digitize and streamline the appointment booking process between doctors and patients.

## Problem Statement
Dental clinics and doctors spend significant time manually managing appointments using paper agendas and books. This leads to scheduling conflicts, missed appointments, and inefficient communication. This system digitizes the process, making appointment management faster, more organized, and accessible from any device.

## Technologies Used
- **Frontend:** HTML5, CSS3, JavaScript
- **Backend:** PHP
- **Database:** MySQL
- **Version Control:** Git & GitHub

## Features

### For Patients:
- Register and Login
- Browse available doctors
- Request appointments
- View appointment history
- Cancel pending appointments
- View in-app notifications

### For Doctors:
- Register and Login
- View appointment requests
- Approve or Reject appointments
- View daily/weekly calendar
- Mark appointments as completed
- Set available working hours
- View patient details
